# 📦 素材库

> **用途**: 存放视频制作素材

---

## 📁 目录结构

```
assets/
├── emojis/          # 表情包（PNG/JPG）
├── backgrounds/     # 背景图（PNG/JPG）
├── music/           # 背景音乐（MP3/WAV）
└── README.md
```

---

## 🎨 添加素材

### 表情包

**放到**: `emojis/` 目录

**格式**: PNG（推荐透明背景）、JPG

**大小**: 建议 512x512 或 1024x1024

**命名建议**: 
- `emoji_happy.png` (开心)
- `emoji_thumbs.png` (点赞)
- `emoji_fire.png` (火)

---

### 背景图

**放到**: `backgrounds/` 目录

**格式**: PNG、JPG

**大小**: 1080x1920（竖屏）或 1920x1080（横屏）

---

### 背景音乐

**放到**: `music/` 目录

**格式**: MP3（推荐）、WAV

**时长**: 建议60秒+（可循环）

**注意**: 使用免费音乐，避免版权问题

---
